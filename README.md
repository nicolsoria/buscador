# Sistema Buscador de Personas

Proyecto desarrollado en PHP y MySQL.

## Funcionalidad

Permite buscar personas por:

- Nombre
- CI

Muestra información como:

- Foto
- Nombre completo
- CI
- Celular
- Dirección

## Tecnologías

- PHP
- MySQL
- Bootstrap
- HTML
- CSS

## Instalación

1. Importar la base de datos `buscador_personas.sql`
2. Ejecutar el proyecto en un servidor local (XAMPP)
3. Abrir en el navegador:

http://localhost/buscador
